Discord Master Bot

Discord Master Bot es un bot de discord que te permite tener un control total de tu servidor, con un sistema de moderación, de música, de economía, de entretenimiento y mucho más.

Estadisticas

Mira las estadisticas de nuestro bot con el comando /botinfo de forma sencilla

Valoraciones

Mira las valoraciones de nuestro bot con el comando /bot modals de forma sencilla

Proyectos

Mira los proyectos de nuestro equipo visitando mi grupo y pagina oficial

Exclusive Proyects

Sistemas de Musica y Paneles Automaticos de diferentes plataformas como Spotify, Colud.

Commands

Command Description
Panel Menus Muestra diferentes menus con informacion y comandos customizados.
Pokemon Comando con gran cantidad de interacciones conforme a la gran serie pokemon.
Bot Info Muestra diferente informacion del bot desde el creador y los servidores actuales
Antispam Diferentes configuraciones de nuestro sistema de antispam integrado
Bienvenidas Sistema de bienvenidas del bot de discord con cards decorativas en los servidores
Despedidas Sistema de despedidas del bot de discord con cards decorativas en los servidores
Estadisticas Muestra un sistema de estadisticas desde miembros a bots dentro el servidor en diferentes canales
Loggers Configura el sistema de logs dentro el servidor con mas de 16+ eventos registrados
Setups Muestra gran cantidad de sistemas que posee el bot de discord para configurar
Sugerencias Muestra el sistema de sugerencias con altas configuraciones posibles dentro del servidor
Tickets Muestra nuestro sistema de tickets altamente configurable
Bot Modals Gran cantidad de formatos para reportes, reseñas y mas acciones del bot de discord
School Libros Muestra los diferentes menus del bot con mas de 10+ guias diferentes cada panel
Github Busca diferentes perfiles de github junto con sus repositorios principales
Diversion Gran cantidad de comandos de acciones, emociones, imagenes dentro de tu servidor de discord
Acciones Genera links de actividades en canales de voz para tu servidor de discord (Algunos piden nitro boost)
